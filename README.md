wlm은 파이썬을 PLC의 Instruction List(IL) 생성기로 사용하기 위한 프로젝트다.
wlm-code는 wlm을 vscode에서 다룰 때 필요한 편의 기능을 제공하기 위한 프로젝트다.
현재는 wlm이 생성한 IL를 문법강조해서 볼 수 있도록 해 준다.
